# Prep(x) Study Planner

Prep(x) is a simple AI-powered study planner that helps you generate a day-by-day schedule for upcoming exams.

---

## Run Locally

### Prerequisites
- Node.js installed

---

### 1. Install dependencies

npm install

---

### 2. Set your Gemini API key

Create a file called `.env.local` in the project root and add:

GEMINI_API_KEY=your_api_key_here

---

### 3. Start the development server

npm run dev

---

### 4. Open the app

Visit:

http://localhost:3000

---

Enjoy studying :)
